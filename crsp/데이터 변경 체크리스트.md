# **Fama-French(1993) 재현 과제 수행 현황**

**기간:** 1963-07 \~ 1991-12

**진행 방향:** CRSP 기반 수익률 처리 \+ Ken French Data Library 기반 포트폴리오/팩터 활용

현재는 Ken French Data Library를 활용해 25개 Size-BE/ME 포트폴리오와 SMB, HML, RM-RF 등 핵심 팩터를 처리하는 코드 구조를 구축했다. 다만 최종 과제 결과는 CRSP 에서 RET, DLRET, PRC, SHROUT 등을 수집한 뒤 수익률 보정과 시가총액 계산을 반영해 재실행할 예정이다.

아래 체크리스트에서 코드 구현 완료는 분석 로직이 준비되었다는 의미이며, 최종 산출 진행중은 CRSP 최종 데이터 반영 후 결과표를 다시 산출해야 한다는 의미이다.

**채권 포트폴리오/채권 변수 분석, Table 2 자기상관계수, Table 9b 채권 절편, Table 9c bootstrap, Table 10 계절성 분석, Table 11 E/P 또는 D/P 포트폴리오 분석은 현재 체크리스트에서 제외**

**(Table 11은 향후 가능하면 수행)**

| **RET** | ⬜ 예정 | WRDS CRSP Monthly Stock File(crsp.msf) |

| **PERMCO** | ⬜ 예정 | WRDS CRSP Monthly Stock File(crsp.msf). 회사 단위 식별자 |
